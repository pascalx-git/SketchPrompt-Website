Quick Installation

Install via Cursor IDE Extensions

For most users, install directly from the Cursor IDE extensions marketplace

1

Open Cursor IDE

Launch Cursor IDE on your computer

2

Open Extensions

Press Cmd/Ctrl + Shift + X or click the Extensions icon in the sidebar

3

Search for SketchPrompt

Type "SketchPrompt" in the search box

4

Install Extension

Click "Install" and restart Cursor IDE (if prompted)

5

Start Sketching

Use Cmd/Ctrl + Shift + P and run "SketchPrompt: New Sketch"

Install from GitHub (For Developers)

For developers who want to build from source or contribute

1

Clone Repository

git clone https://github.com/pascalx-git/SketchPrompt.git
2

Install Dependencies

npm install
3

Build Extension

npm run vscode:prepublish
4

Package Extension

npx vsce package
5

Install in Cursor

Install the generated .vsix file in Cursor IDE

How to Use

Follow these steps to create your first sketch in Cursor IDE

1

Open the Command Palette

Press Cmd/Ctrl + Shift + P in Cursor IDE.

2

Create a New Sketch

Option 1: Type and select SketchPrompt: New Sketch in the Command Palette.
Option 2: Right-click in the file explorer and choose New SketchPrompt Sketch from the context menu.

3

Start Sketching

Use the visual editor to draw, brainstorm, and prompt AI with your ideas!

4

Copy Your Sketch to a Prompt

Option 1: Take a screenshot of your sketch using your system’s screenshot shortcut (e.g., Cmd + Shift + 4 on Mac, Win + Shift + S on Windows).
Option 2: Select the part of your sketch, right-click, choose Copy as PNG, then paste it directly into Cursor chat.
