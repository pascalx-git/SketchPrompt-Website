Quick Installation

Install via Cursor IDE Extensions

For most users, install directly from the Cursor IDE extensions marketplace:

1

Open Cursor IDE

Launch Cursor IDE on your computer

2

Open Extensions

Press Cmd/Ctrl + Shift + X or click the Extensions icon in the sidebar

3

Search for SketchPrompt

Type "SketchPrompt" in the search box

4

Install Extension

Click "Install" and restart Cursor IDE when prompted

5

Start Sketching

Use Cmd/Ctrl + Shift + P and run "SketchPrompt: New Sketch"

Install from GitHub (For Developers)

For developers who want to build from source or contribute:

1

Clone Repository

git clone https://github.com/pascalx-git/SketchPrompt.git
2

Install Dependencies

npm install
3

Build Extension

npm run vscode:prepublish
4

Package Extension

npx vsce package
5

Install in Cursor

Install the generated .vsix file in Cursor IDE

How to Use

Creating a New Sketch

  • Open the command palette (Cmd/Ctrl + Shift + P)
  • Run "SketchPrompt: New Sketch"
  • Or right-click in the file explorer and select "SketchPrompt: New Sketch"
  • A new .sketchprompt file will be created in the SketchPrompt folder

Editing Sketches

  • Open any .sketchprompt file to edit it visually
  • All changes are auto-saved
  • Use the rich drawing tools to create your visual ideas

AI Collaboration

  • Export your sketch as an image
  • Copy and paste directly into Cursor Chat
  • Get AI assistance based on your visual ideas
